Step 2

For APU Student / Staff

Select no of users who will be utilising the discussion room (Remember to include yourself). Then, select booking date where 30 days advanced booking is allowed, followed by start time and end time.

For Public User

Select no of users who will be utilising the discussion room (Remember to include yourself). Then, select booking date where 30 days advanced booking is allowed but only available on weekends, followed by start time and end time.
